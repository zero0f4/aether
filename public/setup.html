🌊 WiFi-pulse · setup

Welkom! Eenmalig configureren — daarna draait WiFi-pulse continu en zie je live je netwerk.

Voor je begint

  1. Log in op je UniFi controller (UDM, UDR, Cloud Key, …).
  2. Ga naar Settings → Admins & Users → Create New Admin.
  3. Maak een local-only user aan met rol Read Only. Geen e-mail nodig.
  4. Onthoud de gebruikersnaam + wachtwoord — invullen hieronder.
💡 Waarom een aparte read-only user? WiFi-pulse heeft alleen leesrechten nodig. Je hoofd-admin-account is niet nodig — meer veiligheid, en als deze user later compromiseert kan hij niets aanpassen.
